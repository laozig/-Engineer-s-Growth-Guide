# C 语言基础

现在你已经搭建好了开发环境，让我们来编写并运行第一个 C 程序。我们将从经典的 "Hello, World!" 开始，并借此了解一个基本 C 程序的结构以及编译和运行它的过程。

## 1. 第一个 C 程序: "Hello, World!"

在你的项目文件夹中，创建一个名为 `hello.c` 的新文件，并输入以下代码：

```c
#include <stdio.h>

// main 函数是每个 C 程序的入口点
int main() {
    // printf 是一个标准库函数，用于向控制台输出文本
    printf("Hello, World!\n");
    
    // 返回 0 表示程序成功执行
    return 0;
}
```

### 程序结构解析

- **`#include <stdio.h>`**: 这是一个**预处理器指令**。它告诉编译器在正式编译之前，把标准输入输出库的头文件 `stdio.h` 的内容包含进来。这个头文件中包含了我们后面使用的 `printf` 函数的声明。
- **`// 这是一个单行注释`**: 注释是用来向阅读代码的人解释程序的。编译器会忽略它们。C 语言还支持多行注释 `/* ... */`。
- **`int main()`**: 这是 `main` 函数的定义。`main` 函数是一个特殊的函数，它是操作系统调用你的程序时的入口点。程序从 `main` 函数的第一行开始执行。
    - `int` 表示 `main` 函数执行完毕后，会向操作系统返回一个整数值。
- **`printf("Hello, World!\n");`**: 这一行调用了 `printf` 函数。
    - `printf` 是 "print formatted" 的缩写，用于向标准输出（通常是你的终端或控制台）打印格式化的文本。
    - `"Hello, World!\n"` 是一个字符串字面量，是传递给 `printf` 函数的参数。
    - `\n` 是一个**转义序列**，代表一个换行符，它会使光标移动到下一行的开头。
    - C 语言中，语句必须以分号 `;` 结尾。
- **`return 0;`**: 这一行表示 `main` 函数执行结束，并向操作系统返回一个值 `0`。在 C 语言中，返回 `0` 通常表示程序执行成功。返回非零值则通常表示程序遇到了某种错误。

## 2. 编译和运行

现在我们有了源代码文件 `hello.c`，需要通过编译器将其转换成可执行文件。

### 方法一：使用命令行

1.  打开你的终端 (Command Prompt, PowerShell, Terminal)。
2.  使用 `cd` 命令切换到 `hello.c`所在的目录。
3.  运行编译器命令。我们将使用 `gcc`（在 Windows 上通过 MinGW 安装，在 Linux 上通过 `build-essential` 安装）或 `clang`（在 macOS 上）。它们的使用方式基本相同。

    ```bash
    // -o hello 指定输出的可执行文件名为 hello
    // hello.c 是输入的源文件名
    gcc -o hello hello.c 
    ```

    如果你使用的是 g++ (C++ 编译器)，命令也类似：
    ```bash
    g++ -o hello hello.c
    ```

4.  执行命令后，如果代码没有错误，你会在当前目录下看到一个新生成的文件。在 Windows 上是 `hello.exe`，在 macOS 和 Linux 上是 `hello`。
5.  运行这个程序：

    -   在 Windows 上:
        ```bash
        hello.exe
        ```
        或者
        ```bash
        .\hello.exe
        ```

    -   在 macOS 和 Linux 上:
        ```bash
        ./hello
        ```

    执行后，你会在终端上看到输出：
    ```
    Hello, World!
    ```

### 方法二：在 VS Code 中运行

VS Code 提供了更集成的体验。

1.  在 VS Code 中打开包含 `hello.c` 的文件夹。
2.  打开 `hello.c` 文件。
3.  点击右上角的"播放"按钮（▶），或者按下 `Ctrl+F5` (Run Without Debugging)。
4.  VS Code 会弹出一个菜单，让你选择编译器。选择你安装的编译器，例如 `C/C++: gcc.exe build active file`。
5.  VS Code 会自动为你执行编译和运行的命令，并在下方的"终端"面板中显示输出结果。

在第一次执行时，VS Code 会在你的项目文件夹下创建一个 `.vscode` 目录，并在其中生成一个 `tasks.json` 文件。这个文件保存了如何编译你的代码的指令，这样你下次再点击"播放"按钮时，就无需再次选择编译器了。

## 3. 注释 (Comments)

注释对于编写可读性强的代码至关重要。C 语言支持两种类型的注释：

- **单行注释**: 以 `//` 开始，直到该行结束。
  ```c
  // 这是单行注释
  int a = 10; // 也可以在代码后面
  ```

- **多行注释**: 以 `/*` 开始，以 `*/` 结束，可以跨越多行。
  ```c
  /*
    这是一个
    可以跨越多行的
    注释。
  */
  ```
